## 使用方法 ##

`Zotero-->编辑-->首选项-->引用-->样式`

<img src="../../src/images/import_csl.gif" style="width: 1000px;" alt="Import CSL" />

## 使用注意 ##

对于英文文献添加至 Zotero 后，需要检查文献信息中是否包含 “语言” 信息，若缺失可以手动添加 “en”，否则会出现英文文献作者中出现 “等” 而不是 “et.al”。
